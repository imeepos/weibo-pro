# Weibo-Pro UI 组件库

欢迎使用 **@sker/ui** 组件库文档。

## 概述

这是基于 [shadcn/ui](https://ui.shadcn.com/) 构建的组件库，采用：

- **React** - UI 框架
- **TailwindCSS** - 样式系统
- **Radix UI** - 无样式原语组件
- **class-variance-authority** - 变体管理

## 可用组件

浏览左侧菜单查看所有可用组件及其用法示例。

每个组件都支持：
- 多种变体（variant）
- 不同尺寸（size）
- 完整的 TypeScript 类型
- 响应式设计
- 暗色模式

## 开始使用

```bash
# 安装依赖
pnpm install

# 启动 Storybook
pnpm dev
```

## 项目结构

- **packages/ui** - 组件源码
- **apps/storybook** - 组件文档和示例
